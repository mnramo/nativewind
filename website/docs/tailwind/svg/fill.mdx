# Fill

## Basic usage

<RNSVG />

Styled components can use the `fill` attribute to pass the fill color to react-native-svg.

```tsc
import { styled } from "tailwindcss-react-native"

const MyStyledSvg = styled({ fill, ...props })
  return (
    <Svg height="100" width="100" {...props }>
      <Rect x="0" y="0" width="100" height="100" fill={fill} />
    </Svg>
  )
}

<MyStyledSvg className="p-4" fill="fill-blue-500" />
```

<Compatibility
  supported={["fill-{n}", "fill-[n]"]}
  none={["fill-inherit", "fill-current"]}
