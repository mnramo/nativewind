# Fill

## Basic usage

<RNSVG />

Styled components can use the `stroke` attribute to pass the stroke width to react-native-svg.

```tsc
import { styled } from "tailwindcss-react-native"

const MyStyledSvg = styled({ strokeWidth, ...props })
  return (
    <Svg height="100" width="100" {...props }>
      <Rect x="0" y="0" width="100" height="100" strokeWidth={strokeWidth} />
    </Svg>
  )
}

<MyStyledSvg className="p-4" stroke="stroke-1" />
```

<Compatibility supported={["stroke-{n}", "stroke-[n]"]} />
